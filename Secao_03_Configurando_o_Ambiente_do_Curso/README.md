# Seção 03: Configurando o Ambiente do Curso 🛠️

## Objetivo da seção 🎯
Configurar o ambiente necessário para executar os exemplos do curso com Apache Spark e Apache Iceberg de forma estável e reprodutível.

## Visão geral da configuração 🔎
Nesta etapa, você prepara as ferramentas e dependências essenciais para acompanhar as práticas do curso, garantindo que os notebooks e scripts funcionem corretamente.

Principais pontos de configuração:
- Preparação do ambiente local de desenvolvimento
- Organização de dependências para Spark e Iceberg
- Validação inicial da execução dos notebooks
- Estrutura básica para iniciar os experimentos práticos

## O que você aprende nesta seção 📘
- Quais componentes são necessários no ambiente
- Como organizar o setup para evitar erros comuns
- Como validar se o ambiente está pronto para as próximas práticas

## Arquivo de apoio 📂
- `Setup.ipynb`: notebook com o passo a passo de configuração da seção

## Conexão com as próximas seções 🔗
Com o ambiente configurado, o curso avança para:
- Primeiros passos práticos com Iceberg
- Exploração de snapshots
- TimeTravel aplicado em cenários reais
