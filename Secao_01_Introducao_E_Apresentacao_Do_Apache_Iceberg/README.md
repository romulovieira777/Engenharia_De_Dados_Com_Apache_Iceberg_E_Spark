# Seção 01: Introdução e Apresentação do Apache Iceberg 🧊

## Objetivo da seção 🎯
Apresentar os conceitos iniciais do Apache Iceberg e mostrar por que ele é relevante em projetos modernos de Engenharia de Dados.

## Visão geral do Apache Iceberg 🔎
O Apache Iceberg é um formato de tabela para data lakes, criado para melhorar confiabilidade, governança e performance em grandes volumes de dados.

Principais capacidades:
- Evolução de esquema sem reescrita completa dos dados
- Versionamento de tabelas por snapshots
- Operações de leitura e escrita mais seguras em ambientes distribuídos
- Suporte a consultas históricas e auditoria
